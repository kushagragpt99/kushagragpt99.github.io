Introduction to tensorflow

Kushagra, Aadil Hayat
slides

Abstract

This talk introduces tensorflow for deep learning and explains its underlying computation methodology.

Date
Jan 4, 2020 12:00 AM
Event
Introduction to tensorflow
Location
IIT Kanpur

This talk was part of Programming Club, IIT Kanpur’s talk on machine learning frameworks. This talk was targeted for sophomores and junior undergraduates with some background on deep learning. It explains dataflow graphs (which represents tensorflow computations) and the basic terminologies and functions. Next, it covers basic implementation in tensorflow through a toy problem on MNIST dataset, also covering the neural network and the dataflow graph for the problem. It also covers the installation procedure of the GPU and CPU-only version.
