A full-day workshop for users who want to get more out of R Markdown (and friends). In this workshop, you will create, publish, and share some beautiful data products. You’ll learn to make:

  • Parameterized reports
  • Version-controlled R Markdown projects
  • Templates within a package
  • Shareable scientific/technical articles, slides, dashboards, and websites
